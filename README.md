## Használat
1. Nyisd meg a fenti linket böngészőben
2. Írd be a hőmérsékleteket 5 zónára, 24 órára
3. C/F gomb: váltás Celsius/Fahrenheit
4. Statisztika: alul automatikusan számol max/min/átlag
5. Nap törlése: gomb + megerősítés
6. Adat mentés: automatikus, böngésző bezárása után is megmarad

## Hőségriadó
40°C / 104°F felett a cella pirosra vált
